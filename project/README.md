# SaaS Boilerplate & WhatsApp Architecture

Selamat datang di repositori SaaS Boilerplate! Proyek ini adalah sistem arsitektur berbasis *multi-tenant* yang ditenagai oleh **Laravel 12 + Inertia React**, serta terintegrasi secara *real-time* dengan **Node.js WhatsApp microservice** dan WebSocket (Laravel Reverb).

## 🚀 Memulai (Developer Onboarding)

Demi menjaga standar dan kualitas arsitektur, seluruh *engineer* atau *developer* baru yang akan bekerja pada repositori ini diwajibkan untuk memulai dari referensi dokumentasi instalasi internal:

1. **[01 - Quickstart (Cara Instalasi Repo)](./docs/01-quickstart.md)**
2. **[00 - Architecture Overview](./docs/00-overview.md)**

Silakan buka folder `docs/` untuk membaca dokumentasi ekstensif tentang otentikasi, otorisasi RBAC, sistem langganan (*subscription*), layanan WhatsApp, panduan struktur test (E2E/Feature), dan lain-lain.
